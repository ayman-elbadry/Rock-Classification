                Architecture
                
À propos du Modèle

            


            

                

                    
🧠

                    
ResNet50

                    
Architecture de deep learning pré-entraînée sur ImageNet, fine-tunée sur notre dataset de roches géologiques pour une classification précise.

                

                

                    
🔬

                    
Grad-CAM

                    
Gradient-weighted Class Activation Mapping permet de visualiser les zones de l'image qui influencent le plus la décision du modèle.

                

                

                    
⚖️

                    
WeightedRandomSampler

                    
Gestion du déséquilibre des classes par échantillonnage pondéré, assurant un entraînement équitable sur toutes les catégories.

                

                

                    
🎯

                    
Transfer Learning

                    
Entraînement en deux phases : d'abord le classificateur seul, puis fine-tuning de toutes les couches avec early stopping.

                

                

                    
📊

                    
Data Augmentation

                    
Rotations, flips, crops aléatoires et jitter de couleur pour augmenter la robustesse et la généralisation du modèle.
